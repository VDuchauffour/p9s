pub enum AppEvent {
    Key(KeyEvent),

- Tick: 30Hz render tick
- Resize: Terminal resize events

- LifecycleComplete: UPID task completion tracking
- LifecycleAction: User-triggered start/stop/reboot

Used in src/lib.rs via tokio::sync::mpsc::unbounded_channel::<AppEvent>().
